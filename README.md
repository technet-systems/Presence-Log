Simple Presence Log (based on Symfony 3.4.7)
============================================

A simple application for creating random students and recording their presence.

What's inside?
--------------

  * [**The Symfony Standard Edition**][1]

  * [**Bootstrap**][2]

  * [**Toastr**][3]

  * [**X-Editable**][4]

Enjoy!
